### 💼 Projetos em Destaque

**[SiteWatch](https://portifolio-theta-two-49.vercel.app/)** 🌐 *(Monitoramento SaaS Autônomo)*  
Plataforma SaaS baseada em microsserviços para monitoramento contínuo de sites. Resolve o problema de downtime utilizando um motor de checagem com cron jobs dinâmicos assíncronos. Exibe o histórico de latência/uptime em gráficos (Recharts) e dispara alertas em tempo real via Webhooks (Discord/Slack) e Resend.
> **Tech Stack:** Next.js 15, React 19, Node.js, Express, PostgreSQL, Prisma ORM, Tailwind CSS v4.

**[CloudChat](https://portifolio-theta-two-49.vercel.app/)** 💬 *(Livechat Real-time de Alta Performance)*  
Plataforma completa de Livechat permitindo transbordo humano de bots de WhatsApp. O sistema foi desenhado para suportar milhares de requisições concorrentes entregando mensagens sem delay de rede utilizando WebSockets. Implementa uma estratégia de banco híbrida para alta disponibilidade.
> **Tech Stack:** React, Node.js, Socket.IO, Redis (Cache em memória), PostgreSQL, Prisma ORM, Docker.

**[App de Enoturismo](https://portifolio-theta-two-49.vercel.app/)** 🍷 *(Mobile com QR Code)*  
Solução mobile intuitiva voltada para o turismo em vinícolas de Pinhal-ES. O aplicativo lê QR Codes espalhados pelos pontos turísticos e entrega detalhes, carrossel de fotos e experiências em tempo real. Estruturado em arquitetura cliente-servidor para iOS e Android. (Projeto aprovado com nota 9,39!)
> **Tech Stack:** React Native, Node.js, TypeScript, MySQL.
